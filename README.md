# Tecnologias:
* Laravel 6
* Angular 9

## Rodando o projeto

Faça o Clone do projeto:

```
    git clone https://github.com/savegdesigner/ImageMuseum.git

```

Instale as dependências:

```
    composer i
    npm i

```

Inicialize o Servidor:

```
    cd backend
    php artisan serve

```

Inicialize a aplicação:

```
    cd frontend
    npm start

```
